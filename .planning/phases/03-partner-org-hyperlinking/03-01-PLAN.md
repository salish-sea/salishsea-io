1. **Import CSV** — static top-level `?raw` import:
```typescript
import partnersRaw from './partners.csv?raw';
```

2. **Partner interface:**
```typescript
interface Partner {
  name: string;
  url: string;
}
```

3. **CSV parser** — `parsePartnersCSV(raw: string): Partner[]`:
   - `raw.trim().split('\n').slice(1)` to skip header
   - `.filter(line => line.trim())` to skip blank lines
   - `.map(line => { const comma = line.indexOf(','); return { name: line.slice(0, comma).trim(), url: line.slice(comma + 1).trim() }; })`
   - Use `indexOf(',')` not `split(',')` — safer for URLs with commas

4. **Module-level constant:**
```typescript
export const partners: Partner[] = parsePartnersCSV(partnersRaw);
```

5. **Per-org injection** — `injectOrgLink(body: string, name: string, url: string): string`:
   - Regex-escape the name: `name.replace(/[.*+?^${}()|[\]\\]/g, '\\$&')`
   - Guard: if `new RegExp('\\[' + escaped + '\\]\\(', 'i').test(body)` — return body unchanged (D-07, PARTNER-06)
   - Single-pass regex combining bracket and plain-name patterns:
     ```typescript
     const re = new RegExp(
       '(\\[' + escaped + '\\](?!\\())|(?<!\\[)\\b(' + escaped + ')\\b(?!\\])',
       'gi'
     );
     ```
   - Replace callback always uses CSV `name` (not matched text) for link text: `() => '[' + name + '](' + url + ')'`

6. **Public API** — `injectPartnerLinks(body: string): string`:
   - Sort partners by name length descending (longest first) to prevent partial matches
   - Reduce over sorted partners calling `injectOrgLink` for each
   - Export as named export

7. Run `npx vitest run src/partner-links.test.ts` — all tests MUST pass (GREEN).
