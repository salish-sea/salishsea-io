**Important context:** Standard `marked.parse('[Name](url)')` produces `<a href="url">Name</a>` WITHOUT `target` or `rel` attributes. To get `target="_blank" rel="noopener noreferrer"` on rendered links, use a custom renderer in the `marked.parse` call. This is consistent with the existing `<a target="_new">` pattern used for attribution links at obs-summary.ts line 174.

**In `src/obs-summary.ts`:**

1. Add import after the existing imports (around line 16, after the `canEdit` import):
```typescript
import { injectPartnerLinks } from './partner-links.ts';
```

2. Replace the body rendering expression at line 176. Current code:
```typescript
unsafeHTML(domPurify.sanitize(marked.parse(body?.replace(/(<br\s*\/?\s*>\s*)+/gi, '\n\n') || '', {async: false})))
```

Replace with:
```typescript
unsafeHTML(domPurify.sanitize(
  marked.parse(
    injectPartnerLinks(body?.replace(/(<br\s*\/?\s*>\s*)+/gi, '\n\n') || ''),
    {
      async: false,
      renderer: {
        link({ href, text }: { href: string; text: string }) {
          return `<a href="${href}" target="_blank" rel="noopener noreferrer">${text}</a>`;
        }
      }
    }
  ),
  { ADD_ATTR: ['target', 'rel'] }
))
```

Three changes from the original:
- Wrap `body?.replace(...) || ''` in `injectPartnerLinks(...)` — pre-processes body text to inject markdown links before marked parses it (per D-03)
- Add `renderer` option with a custom `link` function that outputs `target="_blank" rel="noopener noreferrer"` on all `<a>` tags (per D-05)
- Add `{ ADD_ATTR: ['target', 'rel'] }` as second argument to `domPurify.sanitize()` — preserves `target` and `rel` through sanitization (per PARTNER-04)

The `guard([body], ...)` wrapper and `unsafeHTML(...)` remain unchanged.

**In `src/partner-links.test.ts`:**

3. Add a new `it()` test inside the `describe('injectPartnerLinks', ...)` block for PARTNER-04. This test exercises the full rendering pipeline (marked + DOMPurify) in jsdom:

```typescript
it('preserves target and rel attributes through marked + DOMPurify pipeline', async () => {
  const { marked } = await import('marked');
  const createDOMPurify = (await import('dompurify')).default;
  const domPurify = createDOMPurify(window as any);

  const body = 'Report from Orca Network';
  const processed = injectPartnerLinks(body);
  const html = marked.parse(processed, {
    async: false,
    renderer: {
      link({ href, text }: { href: string; text: string }) {
        return `<a href="${href}" target="_blank" rel="noopener noreferrer">${text}</a>`;
      }
    }
  }) as string;
  const sanitized = domPurify.sanitize(html, { ADD_ATTR: ['target', 'rel'] });

  expect(sanitized).toContain('target="_blank"');
  expect(sanitized).toContain('rel="noopener noreferrer"');
  expect(sanitized).toContain('href="https://orcanetwork.org"');
});
```
