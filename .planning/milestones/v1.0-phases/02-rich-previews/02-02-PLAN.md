  <behavior>
    - Non-bot UA (e.g. "Mozilla/5.0") → handler returns the request object unchanged
    - Bot UA (e.g. "facebookexternalhit") with no ?o= → returns 200 HTML with og:title="SalishSea.io", og:type="website", og:site_name="SalishSea.io", og:url="https://salishsea.io/"; NO og:description or og:image
    - Bot UA with ?o=abc123, Supabase returns vernacular_name="Orca", observed_at="2025-06-03T14:32:00Z", count=3, photo.license="cc0" → og:title="Orca · June 3, 2025", og:description="3 Orcas · 2:32 PM", og:image=photo.src, twitter:card="summary_large_image"
    - Photo license is "cc-by-nc" → og:image="https://salishsea.io/preview.jpg"
    - No photos or all non-open-license photos → og:image="https://salishsea.io/preview.jpg"
    - Supabase returns empty array (not found) → generic preview (og:title="SalishSea.io" only)
    - Supabase fetch throws → return request (fail-open)
    - SSM credentials cached in module scope: second invocation uses same values without calling SSM again
  </behavior>
  <action>
    Create `infra/lib/edge-handler/index.ts` implementing the handler. Follow the pattern from RESEARCH.md exactly. Key implementation rules:

    **Bot detection:** Check `BOT_AGENTS = ['facebookexternalhit', 'twitterbot', 'linkedinbot', 'slackbot', 'discordbot', 'whatsapp', 'telegrambot', 'baiduspider', 'bsky', 'bluesky', 'google-snippet']` — `.toLowerCase()` then `.includes()` check.

    **SSM credentials:** Use module-scoped `let supabaseUrl: string | undefined` and `let supabaseKey: string | undefined`. `SSMClient({ region: 'us-east-1' })`. Parameters: `/salishsea/supabase-url` (plain string) and `/salishsea/supabase-anon-key` (SecureString, `WithDecryption: true`). Cache after first fetch.

    **Supabase query:** `fetch(\`\${url}/rest/v1/occurrences?id=eq.\${encodeURIComponent(occurrenceId)}&select=id,taxon,observed_at,count,photos&limit=1\`, { headers: { 'apikey': key, 'Authorization': \`Bearer \${key}\` } })`

    **Title format (locked):** `\`\${species} · \${date}\`` where date is `Intl.DateTimeFormat('en-US', { month: 'long', day: 'numeric', year: 'numeric' }).format(new Date(occ.observed_at))` — e.g. "June 3, 2025"

    **Description format (locked):** `\`\${count} \${species}s · \${time}\`` where time is `Intl.DateTimeFormat('en-US', { hour: 'numeric', minute: '2-digit' }).format(new Date(occ.observed_at))` — e.g. "2:32 PM". Use `occ.count ?? 1`.

    **License check (locked):** `const OPEN_LICENSES = ['cc0', 'cc-by']`. Check `OPEN_LICENSES.includes(photo.license)` — not just non-null. Only cc0 and cc-by are allowed.

    **Fallback image:** `'https://salishsea.io/preview.jpg'`

    **HTML escaping:** Escape `& " < >` in all dynamic values before interpolating into attribute content.

    **Generic preview tags:** `og:site_name`, `og:type="website"`, `og:url`, `og:title="SalishSea.io"` only — NO og:description, NO og:image.

    **Occurrence preview adds:** `og:description`, `og:image`, `twitter:card="summary_large_image"` (use `<meta name="twitter:card">` not `property`).

    **Entire handler body wrapped in try/catch:** On any error, `return request` (fail-open).

    **Response shape for short-circuit:**
    ```typescript
    return {
      status: '200',
      headers: { 'content-type': [{ key: 'Content-Type', value: 'text/html; charset=utf-8' }] },
      body: buildOgHtml(tags),
    };
    ```

    Do NOT set `environment` on this function — Lambda@Edge does not support env vars.
  </action>
